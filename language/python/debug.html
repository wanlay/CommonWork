# python

Windows

文件 > 首选项 > 设置 > 工作区设置
示例配置

{
    "python.venvPath": "C:/Users/wanlay/.virtualenvs",
    "python.pythonPath": "C:/Users/wanlay/.virtualenvs/flaskVue-zFTujpNl/Scripts/python.exe",
}

TIP

其中python.venvPath是虚拟环境所在的目录
可通过pipenv --venv查询

ctrl + shift + p弹出命令面板，选择解析器select interpreter

配置launch.json

# flask

launch.json

{
    "version": "0.2.0",
    "configurations": [
        {
            "name": "Python",
            "type": "python",
            "request": "launch",
            "stopOnEntry": false,
            "pythonPath": "${config:python.pythonPath}",
            "program": "${workspaceRoot}/run.py",
            "debugOptions": [
                "WaitOnAbnormalExit",
                "WaitOnNormalExit",
                "RedirectOutput"
            ]
        }
    ]
}

run.py

#!flask/bin/python
from app import app

#If debug with vscode,do not add debug=True
#app.run(debug=True)


app.run()
